Running the read simulation pipeline¶

To launch the workflow with default settings:

bash ../ALISEQ/simulate_reads.sh

Modifying simulation parameters¶

Edit the file params.yaml

Uncomment lines as required then adjust settings as desired. Use the default values (provided) as a starting point

It is necesary to set a new “output_tag” for each run, unless “clobber” is also used, in which case it will overwrite previous runfiles of the same name, if they exist.

Note

If the following error is encountered (e.g. after a power failure or network error):

Error

Error: Directory cannot be locked…

run the following cmd before retrying:

snakemake --unlock -s ../ALISEQ/scripts/simulate_reads.snk
